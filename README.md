# Ai_Practice

A practice to implement some famous Neural Network

TAGS: Practice  AI  Neural_Network

## Background
  
  为了研究目前神经网络发展的情况，分别使用Keras和Tensorflow对各种神经网络进行实现

## Install

  直接下载源文件，运行各个框架名文件即可。
  需先安装Tensorflow
